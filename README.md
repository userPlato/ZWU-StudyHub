# 📚 ZWU课程资料共享仓库 | ZWU Notes Repository

> 🌱 由学生自发整理和维护的课程笔记、复习资料与试题合集，致力于打造公平、互助、反对倒卖的学习共享平台。  
> 🌱 A student-initiated, collaborative repository of course notes, exam materials, and study guides, aiming to build a fair and non-commercial academic sharing platform.

---

## 🎯 项目目标 | Project Goals

- ✅ 建立一个系统化的笔记与试题归档库，服务于本校学生。
- ✅ 鼓励跨专业、跨年级的知识共享与学习互助。
- ✅ 抵制有偿倒卖学习资料的行为，维护学习资料公益属性。

- ✅ Build a structured archive of notes and exam materials for students of this university.
- ✅ Encourage inter-major and inter-grade knowledge exchange and academic support.
- ✅ Stand against the commodification of study materials and uphold academic altruism.

## 🤝 如何参与贡献 | How to Contribute

### ✅ 步骤 | Steps
	1.	Fork 本仓库 / Fork this repository
	2.	本地编辑或添加资料 / Add or edit course materials locally
	3.	提交 Pull Request / Submit a Pull Request (PR)
	4.	等待审核并合并 / Wait for review and merge

## 🧾 上传须知 | Submission Notes
	•	请保持资料清晰、有条理，推荐使用 Markdown 编写。
	•	上传前请删除或打码个人信息（如学号、姓名）。
	•	建议对资料来源进行注明（如课程年份、授课老师等）。
	•	若内容非原创，须注明出处并获得原作者许可。
	•	Ensure clarity and structure; Markdown is preferred.
	•	Remove/redact any personal information (e.g., student IDs, names).
	•	Indicate context (e.g., course year, instructor) where possible.
	•	If not original, cite the source and obtain permission when needed.

## 🚫 我们坚决反对 | We Strongly Oppose
	•	将本仓库内容用于任何形式的牟利行为（如笔记贩卖）
	•	私自转载内容而不注明原作者与出处
	•	学术不端行为（如抄袭、代写）
	•	Using this content for any commercial gain (e.g., selling notes)
	•	Republishing without proper attribution
	•	Academic misconduct (e.g., plagiarism, ghostwriting)
